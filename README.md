## About Laravel

Project Point of Sales Laravel 8 + Livewire (SPA)

*First step:
- composer install
- rename .env.example to .env (and configure it)
- php artisan key:generate
- php artisan storage:link
- npm install && npm run dev

*Note
- Database: MariaDB/MySQL (point_of_sales)
